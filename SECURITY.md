## Reporting a Vulnerability

We take the security of Mnemora seriously. If you believe you have found a security vulnerability in Mnemora, please report it to us as described below.

### How to Report

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them by opening a [Security Advisory](https://github.com/AIkaptan/Mnemora/security/advisories/new) or emailing the maintainers directly if an email is provided in the profile.

You should include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Any relevant logs or screenshots.

### Response Timeline

We will try to acknowledge your report within 48 hours. We will keep you updated on our progress as we investigate and fix the reported issue.

### Disclosure Policy

We ask that you give us a reasonable amount of time to fix the issue before identifying it publicly. We will notify you when the fix is released.

Thank you for helping keep Mnemora safe!
